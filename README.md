## Setup
1. Clone the repository.
2. From your linux terminal, navigate to the local repository folder.
3. Run `./vendor/bin/sail up`
4. Open bash in the created docker image
5. Run `php artisan migrate`
6. Visit http://localhost
7. Thanks!
